# AudioBook - PDF to Audiobook Converter

A Python-based text-to-speech application that converts PDF files into audiobooks with customizable voice and speed settings.

## Features

- **PDF Text Extraction**: Efficiently extracts text from PDF files with progress tracking
- **Multiple Voice Options**: Support for different system voices (gender, accent variations)
- **Adjustable Speed**: Speech rate control from 50 to 300 words per minute
- **Progress Tracking**: Real-time feedback during extraction and conversion
- **Error Handling**: Comprehensive error management with user-friendly messages
- **CLI Interface**: Simple command-line interface for easy use

## Prerequisites

- Python 3.7 or higher
- pip (Python package manager)

## Installation

```bash
git clone https://github.com/Lennox2739/AudioBook.git
cd AudioBook
pip install PyPDF2 pyttsx3
```
